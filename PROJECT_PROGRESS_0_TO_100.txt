PROJECT GOAL
------------
Build one premium mobile-first platform serving:
- B2C: factory to personal/retail customer.
- B2B: factory to approved business/wholesaler.
- Owner/admin: catalogue, prices, photos, stock, applications, requests and website content without code editing.

CONFIRMED PRODUCT DIRECTION
---------------------------
Status: APPROVED

- Premium public landing page and searchable catalogue.
- Separate Personal/B2C and Business/B2B experiences.
- Phone-number and OTP authentication.
- Supabase Test OTP for development and staging.
- First-time customers choose Personal or Business only after OTP verification.

DEPLOYMENT FAILURE 002 — RECURSIVE OPENNEXT BUILD
-------------------------------------------------
Observed:
- Deployment ran longer than 20 minutes.
- Repeated “OpenNext — Cloudflare build” headings.
- Repeated `$ opennextjs-cloudflare build` lines.

Cause:
- package `build` script pointed to OpenNext.
- OpenNext internally invoked the package build script, causing recursion.

Fix:
- `build` restored to `next build`.
- Added `cf:build` for OpenNext.
- Cloudflare Build command changed to `npm run cf:build`.
